Nina

Host install (Docker Compose + Docker Hub images).

curl -fsSL https://motirolabs.com/nina/install.sh | bash
  1. Edit /opt/nina/.env (Telegram token, owner allow-list, DB password).
  2. Start: nina up

Optional immediate start: curl -fsSL https://motirolabs.com/nina/install.sh | INSTALL_START=1 bash
